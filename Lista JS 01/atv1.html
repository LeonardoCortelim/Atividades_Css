O que é JavaScript?

O JavaScript foi criado em 1995 por Brendan Eich com o objetivo de tornar as páginas web interativas. Ele surgiu como uma linguagem leve e dinâmica que pudesse ser executada diretamente no navegador, permitindo animações, validações de formulários e interações em tempo real.

A execução no navegador ocorre no lado do cliente, ou seja, no dispositivo do usuário. Já no servidor, o JavaScript pode ser usado com tecnologias como Node.js, permitindo que o código seja executado antes de ser enviado ao navegador, geralmente para manipular dados, acessar bancos de dados ou lidar com autenticação.

O conceito de sandbox no navegador refere-se a um ambiente isolado onde o JavaScript é executado. Isso impede que scripts maliciosos acessem diretamente o sistema do usuário, garantindo maior segurança.

Alguns objetos globais importantes em JavaScript são:

  • document: permite acessar e manipular o conteúdo HTML da página.
  • console: usado para registrar mensagens e depurar o código.
  • alert: exibe mensagens em pop-ups para o usuário.
